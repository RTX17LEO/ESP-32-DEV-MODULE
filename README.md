 ESP32 Dev Board Project

The ESP32 Dev Module is a powerful microcontroller with built-in Wi-Fi and Bluetooth, widely used for IoT and embedded applications.  
This project includes the schematic and PCB design of an ESP32 Dev Module created using KiCad.  
The schematic defines connections for power, EN pin, and basic components like resistors and capacitors.  
A 2-layer PCB is designed with proper routing and clearance rules followed.  
KiCad is used for schematic capture, footprint assignment, and PCB layout, and Gerber files are generated for manufacturing.
